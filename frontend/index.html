02Why OPG Vision

Built for forensic officers,
not data scientists.

No AI expertise required. Upload an OPG, fill in the case details, and download a signed forensic report — in under three minutes. Every analysis is logged, versioned, and audit-ready.

F.01 / Capture
OPG upload & instant inference

Drag and drop JPG, PNG, or DICOM. The ResNeXt-50 model returns a point estimate with bootstrap confidence interval in under three seconds.

F.02 / Report
Court-ready PDF generation

Every analysis emits a structured A4 report — case ID, officer, methodology, disclaimer, and result classification — ready to attach to legal proceedings.

F.03 / Threshold
Borderline alerting at 18 yrs

When the confidence range crosses the legal minority threshold, OPG Vision automatically flags the result and requires specialist sign-off.

F.04 / History
Versioned case dashboard

Every analysis is logged with model version, timestamp, and officer attribution. Filter, search, and re-download without re-uploading the image.

F.05 / Privacy
Zero image retention

OPG images are never persisted. Only report metadata is stored, tied to your organisation under JWT-authenticated access. PDPA Act 2024 compliant.

F.06 / Provenance
UM-patented & UMCIE licensed

Model IP licensed from Universiti Malaya via UMCIE. Trained and validated on a Malaysian multi-ethnic OPG dataset spanning ages 10–70.
