# ResNeXt-50 CNN Dental Age-Estimation System

**Patent:** PI 2025003749 (UMCIE)

## Overview

An AI diagnostic platform that estimates chronological age from dental OPG (orthopantomogram) X-rays using a ResNeXt-50 convolutional neural network. Built as a full-stack system and delivered as a fully functional prototype within a competition timeline.

## Problem Statement

Brief description of why age estimation from dental X-rays matters — e.g., forensic age estimation for law enforcement / immigration verification where documentation (birth certificates, passports) is unavailable or unreliable.

## Key Features

- Chronological age estimation from dental OPG X-ray images using ResNeXt-50 CNN
- End-to-end diagnostic pipeline: image upload → preprocessing → model inference → results
- Full-stack web platform for clinical/institutional use
